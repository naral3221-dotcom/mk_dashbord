# /start Command - 세션 시작

> **사용 시점**: 새 세션 시작 시 또는 작업 재개 시

## 실행 절차

### 1. 컨텍스트 파일 읽기 (필수)
```
Read: .claude/CONTEXT.md      → 현재 상태
Read: .claude/ROADMAP.md      → 로드맵 확인
Read: .claude/logs/{오늘}.md  → 오늘 작업 내역
Read: CLAUDE.md               → 프로젝트 규칙
```

### 2. 상태 보고 형식
```
📍 컨텍스트 복원 완료

## 현재 상태
- **Sprint**: Sprint X - Name
- **마지막 작업**: [작업명]
- **상태**: [완료/진행중/블로커]

## 다음 태스크
| # | Task | Status |
|---|------|--------|
| X.1 | 태스크명 | ⬜ Todo |
| X.2 | 태스크명 | ⬜ Todo |

## 대기 중인 결정
- [있으면 표시]

어떤 작업을 진행할까요?
```

### 3. 로그 파일 생성
오늘 날짜 로그 파일이 없으면 생성

---

## 사용 예시

**사용자 입력**:
```
/start
```

또는

```
세션 시작
```

또는

```
컨텍스트 복원
```
